# Sorteador de Números

### Tecnologias utilizadas
- HTML
- CSS
- JavaScript
### Modo de usar:
1. Escolha a quantidade de números a ser sorteado
2. Defina o range
3. Clique em sortear

### Informações recebidas: 
- Número sorteado
- Data e hora do sorteio
